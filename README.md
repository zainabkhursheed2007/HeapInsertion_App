# HeapInsertion_App
This project is a Min Heap Insertion Tool developed as part of my Data Structures and Algorithms coursework. It combines practical C++ implementation with a modern Windows Forms (GUI) to visualize how heaps manage data and to explore advanced research regarding insertion efficiency. 
While most textbooks define Min Heap insertion as O(\log n), this project dives deeper. Based on the research paper “Repeated Random Insertion into a Priority Queue” (Bollobás and Simon), this application highlights that the average-case insertion time is actually a constant ≈ 1.7645 comparisons.This project proves that in real-world scenarios, heap insertion behaves much closer to O(1) than the theoretical worst-case.
Built With:
Language: C++
Framework: .NET Windows Forms (CLR)
IDE: Visual Studio
Implementation: Array-based Min Heap
